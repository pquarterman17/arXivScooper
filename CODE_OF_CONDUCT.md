# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**, in full,
as its code of conduct. The complete text and translations are
maintained at:

- <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>
- <https://www.contributor-covenant.org/version/2/1/code_of_conduct/translations>

By participating in this project — opening issues, submitting pull
requests, commenting on either, or any other interaction — you agree to
abide by the standards described there.

## Reporting

To report a concern:

- For most issues, [open a GitHub issue](https://github.com/pquarterman17/arXivScooper/issues/new)
  and describe what happened. Issues are public by default — keep
  identifying details out of the title and post-body if you want a more
  private channel first.
- For sensitive or confidential reports (harassment, doxing, anything
  you want kept off the public timeline), use **GitHub's private
  vulnerability reporting** at
  <https://github.com/pquarterman17/arXivScooper/security/advisories/new>.
  It's labelled "security advisory" but the form works for any
  privately-routed message to the maintainer; the conversation stays
  visible only to you and the maintainer until/unless explicitly
  published.

Reports are reviewed and acted on promptly and confidentially. For
security vulnerabilities specifically, see `SECURITY.md`.

## Enforcement

The project maintainer is responsible for enforcement and will follow
the Community Impact Guidelines outlined in section 5 of the
Contributor Covenant linked above.

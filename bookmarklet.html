📎 Bookmarklet Setup

Add papers to your SCQ database with one click

Installation

Step 1: Drag the button below to your browser's bookmarks bar.
📎 Add to SCQ
Step 2: When you're on an arXiv abstract page or a webpage with citation metadata, click the bookmark to send it to your database.

What It Does

  • Detects arXiv page URLs (arxiv.org/abs/...)
  • Extracts DOI from <meta> tags
  • Captures page title, authors, and abstract
  • Sends metadata to the SCQ database
  • Shows success/error notification
Note: The bookmarklet requires the SCQ database server to be running at http://localhost:8080. Make sure you've launched it with START.bat or python -m scq serve.

Workflow

For arXiv papers: Click the bookmark on the abstract page to queue it for full processing. The system will fetch the PDF and extract metadata automatically.

For journal articles: If the page has a DOI in its metadata, the bookmark will capture it. You can then use the database's search or import tools to fetch the full paper details.

Processing: After clicking the bookmark, the paper is queued in the inbox/ folder. Use the "Import" button in the database's More menu to process queued papers.
